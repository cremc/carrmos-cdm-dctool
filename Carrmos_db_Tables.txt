NOTES:
1. Text in parentheses after field name is only an explainer text. It is not part of column name.
2. A few domains that had not been mentioned previously, have been added here (Ranking, Finance). 
3. There is one large section of the database (the User domain that captures the details of a career aspirant) which I have not added here, as they are not required in the current scope of the DC tool. The User of the DC tool is an internal user, not our customers. They are our employees or contractors. If the need arises later in the DC tool development, we can add the customer user in the Qued database.
4. The database in the current form has been designed for the Initial schema and for Indian geographical region. Minor changes in the database may be required (addition/removal of tables or columns) as we progress.
